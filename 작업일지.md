## 2. 사주 계산 엔진 구현

### 핵심 로직 (`src/lib/saju/`)

| 파일 | 역할 |
|------|------|
| `index.ts` | 전체 사주 분석 메인 함수 (`analyzeSaju`) — 음력→양력 변환 포함 |
| `fourPillars.ts` | 사주 사기둥(년·월·일·시주) 계산 |
| `elements.ts` | 오행 밸런스 계산, 용신 추정 |
| `tenGods.ts` | 십신(십성) 계산 |
| `interactions.ts` | 합충형파해 분석 (천간합/충, 지지삼합/방합/육합/충/형/파/해) |
| `specialStars.ts` | 신살 분석 |
| `daeun.ts` | 대운 계산, 세운/월운 계산, 순역행 판단 |
| `coreSummary.ts` | 격국 + 십신 종합 핵심 요약 |

### 상수 테이블 (`src/lib/constants/`)

| 파일 | 역할 |
|------|------|
| `stems.ts` | 천간 데이터 (한글, 한자, 오행, 음양) |
| `branches.ts` | 지지 데이터 (한글, 한자, 띠, 오행, 음양, 장간) |
| `lookupTables.ts` | 십성·12운성·공망 조회 테이블 |
| `solarTerms.ts` | 절기 데이터 |

### 타입 정의 (`src/types/saju.ts`)

- `Pillar`, `FourPillars` — 사주 기둥 구조
- `DaeunCycle`, `AnnualLuck`, `MonthlyLuck` — 대운/세운/월운
- `Interaction` — 합충형파해
- `SpecialStar` — 신살
- `ElementBalance` — 오행 밸런스
- `GongMang` — 공망
- `SajuResult` — 전체 분석 결과 통합 인터페이스

## 5. 페이지 구현

### 입력 페이지 (`src/app/page.tsx`)

- 성별 선택 (남/여)
- 양력/음력 전환 (음력 선택 시 평달/윤달 옵션 표시)
- 생년월일 입력 (YYYYMMDD 숫자만)
- 태어난 시간 입력 (HHMM, 선택사항)
- 다크 테마 UI (배경 `#0D0D14`, 보라-핑크-블루 그라디언트)

### 결과 페이지 (`src/app/result/page.tsx`)

- URL 쿼리 파라미터로 입력값 수신
- `analyzeSaju()` 호출하여 전체 분석
- 섹션 순서: 사주팔자 → 핵심요약 → 일간분석 → 십신 → 오행밸런스 → 신살 → 대운 → 세운 → 월운
- Suspense로 로딩 상태 처리
- sticky 헤더, "다시 입력" 네비게이션

- 30개+ 신살에서 **주요 12신살만** 유지

- `data/interpretations/specialStars.ts` 해석 데이터: 12개만 남기고 제거
- 대운/세운/월운 `getSpecialStarsForBranch` 함수도 12개만 반환하도록 정리
